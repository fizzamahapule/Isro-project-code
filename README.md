# 🌍 AI-Powered Digital Twin of India's Climate

An AI-powered digital twin platform that models and analyzes India's climate using machine learning and interactive visualizations. The application enables users to explore climate data, generate AI-driven insights, and better understand environmental trends across different regions of India.

## 🚀 Features

- 🌦️ Climate data visualization
- 🤖 AI-powered climate analysis
- 📊 Interactive dashboard
- 📍 Regional climate insights
- ⚡ Fast and responsive frontend
- 🔗 Python backend API

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git
cd <repository-name>
```

### Backend Setup

```bash
cd backend

python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux/macOS
source .venv/bin/activate

pip install -r requirements.txt
```

Run the backend:

```bash
python app.py
```

*(Replace with your actual startup command if using FastAPI, Flask, etc.)*

### Frontend Setup

```bash
cd frontend

npm install
npm run dev
```

Open:

```
http://localhost:3000
```
